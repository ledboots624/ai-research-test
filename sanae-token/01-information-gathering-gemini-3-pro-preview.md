# Sanae Token (SANAET) 総合リサーチレポート

## 1. 基本情報
- **名称**: Sanae Token (SANAET)
- **概要**: 日本の政治家・高市早苗氏をテーマにしたミームコイン。「Japan is Back」プロジェクトの一環として、民主主義のアップデートや国民参加型プラットフォームの構築を目指すという名目で発行されました。
- **ブロックチェーン**: Solana (メイン)、BSC (流動性は極めて低い)
- **発行日**: 2026年2月25日
- **公式サイト**: [https://japanisbacksanaet.jp/](https://japanisbacksanaet.jp/) (現在は機能停止または閉鎖状態)
- **コントラクトアドレス (Solana)**: `cuxlbyazac6x89me2unfukrpy1h775rpvce3i5l9ztbi`

## 2. 開発チーム・背景
- **発行主体**: NoBorder DAO
  - YouTube番組「NoBorder」および同名のWeb3コミュニティが母体。
- **主要メンバー**:
  - **溝口勇児氏** (実業家): NoBorder DAOの発起人・運営責任者。
  - **藤井聡氏** (京都大学大学院教授): 政策アドバイザーとして協力。トークン発行への直接関与は否定しつつも、DAOの理念設計に寄与。
- **政治家との関係**:
  - **高市早苗氏は一切関与していません。**
  - 2026年3月、高市氏は自身のX (旧Twitter) で「全く関知していない」「名前や肖像の無断使用」であるとして、公式に関係を否定しました。

## 3. トークノミクスと市場動向
- **発行総数**: 10億枚
- **アロケーション**: 
  - エコシステム: 65%
  - コミュニティ (Airdrop): 20%
  - 流動性: 10%
  - チーム: 5%
- **市場価格**: 
  - 発行直後に急騰し、時価総額は一時2,700万ドル (約40億円) に達しましたが、高市氏の関与否定後に暴落。
  - Solana版の現在価格は約 $0.016 前後で推移していますが、ボラティリティが極めて高い状態です。
- **上場取引所**:
  - **CEX (中央集権取引所)**: Bybit, MEXC, Gate.ioなどの主要取引所への上場実績はありません。
  - **DEX (分散型取引所)**: Raydium (Solana), PancakeSwap (BSC) などで取引されていましたが、現在は流動性が枯渇しつつあります。

## 4. ロードマップと最新状況
### 当初の計画
- 民主主義への参加インセンティブとしてのトークン活用
- コミュニティ主導の政策提言プラットフォーム構築

### 現在の状況 (2026年3月時点)
- **プロジェクト中止**: 高市氏の否定および金融庁による調査開始を受け、NoBorder DAOはプロジェクトの事実上の中止を発表しました。
- **発行停止**: 2026年3月4日に新規発行を停止。
- **補償プロセス**: 3月4日時点の保有者 (スナップショット) を対象とした補償を検討中としていますが、具体的な返金方法や時期は未定です。
- **名称変更**: プロジェクト存続のためにトークン名称の変更も検討されましたが、信頼失墜により実現性は不透明です。

## 5. リスク評価と結論
### リスク要因
- **規制リスク**: 金融庁が資金決済法違反（無登録交換業など）の疑いで調査を行っており、法的な存続が危ぶまれています。
- **信用リスク**: テーマとした政治家本人からの明確な否定により、プロジェクトの根幹となる信用が崩壊しています。
- **投資リスク**: 流動性が低く、プロジェクト自体が中止されているため、価値がゼロになる可能性が極めて高いです。

### 結論・推奨事項
**Sanae Tokenへの投資や関与は推奨されません。**
プロジェクトは既に破綻しており、現在は事後処理（補償対応）のフェーズにあります。市場での購入は極めてハイリスクであり、既存の保有者は公式発表（NoBorder DAOのXアカウント等）での補償情報を待つことが唯一の選択肢となります。
