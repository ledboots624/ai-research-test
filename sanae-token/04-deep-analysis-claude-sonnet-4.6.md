### 1.2 資金決済法違反：最大の法的リスク

**根拠となる規定**

日本の資金決済法（第2条・第63条の2）では、**暗号資産交換業を国内居住者向けに「業として」行う場合、金融庁への登録が必須**である。

**SANAETに該当する違反疑惑**

```
① NoBorder DAOは暗号資産交換業者として金融庁に未登録
② 日本国内ユーザー向けに積極的なSNSマーケティング・勧誘を実施
③ DEX（Raydium）上での取引促進行為が「業として」の仲介に該当する可能性
④ Airdrop配布行為自体が「交換業」とみなされるリスク
```

**法定制裁レベル**

| 違反類型 | 制裁内容 |
|--------|--------|
| 資金決済法違反（無登録） | **3年以下の懲役または300万円以下の罰金**（法人は1億円以下） |
| 金融商品取引法違反（適用される場合） | **10年以下の懲役または3,000万円以下の罰金** |
| 業務改善命令・業務停止命令 | 行政処分（既に事業停止状態であるため実効性は限定的） |

> 出典：産経ニュース（金融庁調査報道, 2026/03/04）、Phemex News（FSA刑事捜査検討, 2026）、CoinChoice（徹底解説, 2026）

### 1.4 保有者への規制リスク波及

保有者は直接の法的責任は低いが、以下の点で二次的影響を受ける可能性がある：

- **税務上の取り扱い**：損失が発生した場合でも、取得・売却の申告義務あり（雑所得扱い）
- **補償受取時の課税**：NoBorder DAO補償が「損害賠償金」か「所得」かで課税関係が異なる
- **凍結リスク**：当局による資産凍結命令が出た場合、DEX上の流動性が影響を受ける可能性

**価格サイクルの実態**

```
2026年2月25日  発行開始
      ↓
2026年3月初旬  時価総額 ~$27M（ピーク）
      │
      │ 高市首相否定声明（3月2日）
      ↓
数時間以内     -50〜80% 急落
      │
      │ 金融庁調査報道（3月4日）
      ↓
現在           $3.9M〜$16M 水準で乱高下
              日次出来高：$6,000〜$9,500（枯渇状態）
```

**価格崩壊を加速させた構造的要因**

| 要因 | 詳細 | 影響度 |
|-----|------|------|
| 信用根拠の崩壊 | ナラティブ（公認コイン）の全面否定 | 🔴 致命的 |
| 流動性の薄さ | プール規模$285K〜$575K → 小さな売りで大幅下落 | 🔴 高 |

| ネガティブ報道の連鎖 | 金融庁・首相否定・詐欺疑惑が同時進行 | 🔴 高 |


> 出典：Invezz on-chain analysis (2026/03/03)

---

### 3.2 中央集権化が生む具体的リスク

**① 価格操作リスク**

上位3ウォレットが保有量の1〜2%を市場放出するだけで、流動性プール（$285K〜$575K）に対して数十倍の売り圧力が発生する。これは事実上の**価格支配**であり、小口投資家が太刀打ちできない構造である。

**② ガバナンス乗っ取りリスク（仮にDAO実装された場合）**

```
仮定: ガバナンス機能が実装された場合
  → 上位3ウォレット = 60%の投票権
  → 残り950ウォレット = 最大40%の投票権
  → 単独過半数達成が容易 → ガバナンスは形骸化
```

ただし現状ガバナンス機能は未実装であり、この問題は理論上の課題に留まる。

**③ 「エコシステム枠」65%の構造的問題**

| 問題 | 詳細 |
|-----|------|
| ロック無し | 65億枚相当が即時市場放出可能な状態 |
| 運営裁量 | 放出タイミングは完全に運営側が制御 |
| 透明性ゼロ | エコシステム利用目的・条件が非公開 |
| 実質支配 | DAOを標榜しながら実態は中央集権的統制 |

**④ 「分散型」表記との矛盾**

NoBorder DAOは「DAO（分散型自律組織）」を名乗るが、以下の実態は**中央集権的管理**を示す：

- 代表者（溝口勇児氏）が単独で謝罪・補償・名称変更を決定
- オンチェーン投票によるガバナンス決議プロセスが存在しない
- スナップショット日時・補償額が運営の一存で決定

> 出典：NOVAIST（NoBorder謝罪記事, 2026）、J-CAST（プロジェクト中止記事, 2026）

## 6. 結論と推奨事項

### 6.1 リスクの本質的特徴（要約）

1. **規制リスクは「リスク」ではなく「現実」となっている**：金融庁調査は進行中であり、刑事捜査移行報道もある。これは「将来のリスク」ではなく**今まさに発現しているリスク**である。

2. **市場ボラティリティはもはや「双方向」ではない**：出来高が$6,000程度に枯渇した市場での「上昇」は非常に限定的・短命であり、出口機会として機能しない。

3. **中央集権化は構造的に解決不可能**：プロジェクトが終了した状態では、65%の運営保有トークンは「市場に放出されるか否か」だけが問題であり、ガバナンスによる改善は不可能。

4. **持続可能性は終了した**：補償の有無を問わず、SANAETは新規の価値創造を行う基盤を失っている。現在の価格は純粋な投機的残余価値に過ぎない。

---

### 6.2 ステークホルダー別推奨アクション

**現在の保有者へ**
- 追加購入は**絶対に行わないこと**（上昇余地より下落余地が圧倒的に大きい）
- 3月4日スナップショット対象者は**補償申請情報を公式X（@NoBorderDAO等）で追跡**
- 損失が大きい場合は、**弁護士相談を検討**（パブリシティ権侵害・詐欺的勧誘として民事請求の可能性を含む）
- 保有・売却にかかわらず**税務申告の準備**（雑所得）

**業界・規制当局への示唆**
1. **著名人名義トークンへの明示的な本人同意要件の法整備**（景品表示法の改正or暗号資産法の新設条項）
2. **ミームコイン発行者への最低限のベスティング・LP固定要件**の義務化
3. **DAOの法的人格の整備**：溝口氏個人の刑事責任が問われる場合、DAOという組織形態が責任回避の隠れ蓑になるリスクへの対処

### 6.3 最終警告

> SANAETへの追加投資は、  
> **プロジェクトが正式終了し、規制当局の調査が進行中で、補償も未確定な状態において**、  
> 唯一の「上昇要因」として機能し得るのは**投機的な話題性の再燃のみ**である。
